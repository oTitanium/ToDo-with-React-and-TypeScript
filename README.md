# ToDo-with-React-and-TypeScript

**Description**
This is a To-Do web app for tracking and displaying a list of to-do
items.

**Built With**
- React
- TypeScript
- Vite
  
**Installing**
- Install Node.js and npm before running the code.
  

**Executing program**

- cd {filename}
- npm install
- npm run dev
